Features
  • Multi-protocol support
  • Group-based management
  • Base64 encoded output
  • Public subscription API
  • Secure access tokens
  • Easy to use interface
Tip: Each subscription group gets unique URLs for different clients:
  • /sub/{token} - V2Ray (Base64 encoded)
  • /sub/{token}?target=clash - Clash (YAML format)
  • /sub/{token}?target=raw - Raw node list
